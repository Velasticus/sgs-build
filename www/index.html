Project Darkstar Build Utilities
The sgs-build project contains various tools and utilities that help facilitate building Project Darkstar and its components. This includes the Project Darkstar master POM (the parent pom.xml used with Maven builds), checkstyle and findbugs configurations, and license file resource bundles. This site provides access to the source code repository, issue tracker, and mailing lists for this project. Please see http://www.projectdarkstar.com for more information.
Source code
The publicly accessible Subversion repository contains all of the source code for the sgs-build project. In addition to the java.net Subversion browser linked above, the repository may also be viewed using the FishEye browser.
Issue tracker
Issuezilla is used to track all issues with sgs-build including defects, enhancements, and outstanding tasks.
Mailing Lists
Subversion commit notifications and Issue tracker issue change notifications are sent to separate mailing lists for the sgs-build project. For development discussions and general announcements, please join the dev and announce lists for the games-darkstar parent project.
